# Tech Stack

## Frontend

- **React** with **TypeScript**
- **Tailwind CSS** for styling
- **Vite** for build tooling

## Backend

- **Node.js** with **Express** and **TypeScript**
- **Prisma** ORM
- **Database sessions** for authentication (server-side session storage)

## Database

- **PostgreSQL**

## AI

- **Claude API** — ticket classification, summaries, suggested replies, and response generation

## Email Integration

- **SendGrid** or **Mailgun** — inbound email parsing (webhooks) and outbound replies

## Deployment

- **Docker** containers deployed to a cloud provider
